1.0.0 - what is Ωpm today?

Dependency checker

Check for updates from any of your dependencies with one command: 

omegapm status

Secure

OmegaPM checks that the PGP signature on each message matches the public key in the module downloaded to your computer.

That means that OmegaPM, NPM, and other third parties cannot fake messages after a package is downloaded.

NodeJS and Ruby

This release is being made available for NodeJS and Rails apps.

Installing

CLI

npm install omegapm -g

package.json

Add a public key: 

              {
                "name": "omegapm",
                "publicKey": "key_file.asc"
              }

1.1.0+ - what's next?

Open Standards

Ωpm should be able to read messages from a git repo, if their commits are signed with 'git commit -S'.

Beating Metadata

Padding and combining messages would keep third parties from knowing what you're reading.

HTTPS and Tor

I launched before becoming HTTPS-ready. Got to work on that.

Ωpm should have a hidden service on Tor so package developers and users can communicate without being in the open.
